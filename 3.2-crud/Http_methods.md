GET /products/ - получить список всех
GET /products/1/ - получить конкретный объект
POST /products/ - создать новый объект, запрос отправляется со словарем
PUT /products/1/ - заменить полностью конкретный объект, запрос отправляется со словарем
PATCH /products/1/ - изменить частично конкретный объект, запрос отправляется со словарем
DELETE /products/1/ - удалить конкретный объект
